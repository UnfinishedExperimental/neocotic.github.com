mediad is an online directory which allows users to list their media collection as well as the ability to share it with their friends.

The project was originally developed in Java and targeted for the Google App Engine. However, this plan was dropped as I wanted to increase the scope of this project by rewriting it in PHP as well as changing the whole approach.

This has returned to the planning stage and development has not yet restarted. This is a massive undertaking for me and I want to make sure I get it right as I've had this idea for almost a decade and I'm very passionate about it.
