Template is an extension for Google Chrome that provides quick and easy access to lots of information about the current page and environment;

all links on a page, author, browser version, cookies, description, directory, encode/decode anything, favicon URL, file, formatted date & time, fragment, host name, keywords, last modified date & time, location, operating system, options, page dimensions, parameters, path, plugins, port, protocol, query string, referrer, relative, screen resolution and other information, scripts, segments, selected links/text, shortened URL, stats, style sheets, title, and the URL itself of course

Just to name a few...

Better yet, using a unique template system it allows users to easily create their own templates making the possibilities endless and includes a built-in step-by-step guide to help you write your own. These predefined templates are also provided to help you get started;

  • URL - Copies the URL of the current tab
  • Short - Copies the shortened URL (using any of the supported services)
  • Anchor - Copies an HTML anchor tag (<a>) with the URL and the tab's title
  • Encoded - Copies the encoded URL (to be safely added directly in to an HTTP query)
  • Markdown - Copies a Markdown inline link with the URL and the tab's title
  • BBCode - Copies a BBCode URL tag ([url]) with the URL and the tab's title

So with a click of a button you can add all this information directly to your system's clipboard without any formatting. This saves you so much time with those tedious tasks and it's certainly any testers best friend.

With support for keyboard shortcuts and context (right-click) menus access to your favourite templates is even quicker and they are all easily manageable thanks to Template's unique and simple options page.

URL Shorteners

The following URL shortener services are supported by this extension;

  • bit.ly - Link to your bit.ly account to allow tracking on your timeline
  • goo.gl - Login to your Google account to allow tracking on goo.gl
  • YOURLS - Link to any custom YOURLS installations

Supported Extensions

  • IE Tab
  • IE Tab Classic
  • IE Tab Multi (Enhance)
  • Mozilla Gecko Tab
