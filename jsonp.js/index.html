jsonp.js is a library that provides simple JSONP support for JavaScript.

Get

Send the parameters to the provided URL and then handle the response with a function which is called under a specific context.

{% highlight javascript %} JSONP.get(url[, data][, callback][, context]) {% endhighlight %}

Configuration

The behaviour of jsonp.js can easily be configured further to suit your needs or that of the web service you're calling.

{% highlight javascript %} JSONP.callbackName = 'callback' {% endhighlight %}

Change the name of the callback parameter declaring the name of the callback function.

{% highlight javascript %} JSONP.timeout = 0 {% endhighlight %}

The number of milliseconds before a request times out. Requests won't timeout if the value is below 1.

Miscellaneous

The jsonp.js API exposes some properties and methods which may only be useful to users in special cases.

{% highlight javascript %} JSONP.noConflict() {% endhighlight %}

Relinquish jsonp.js' control of the JSONP global variable. If another library uses this variable calling this method will reassign it back to that library.

{% highlight javascript %} JSONP.VERSION {% endhighlight %}

The current version of jsonp.js.
