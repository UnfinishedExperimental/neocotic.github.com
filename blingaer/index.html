blingaer is a URL shortener engine powered entirely by Java and designed to run only on the Google App Engine that allows anyone to host their very own service.

Inspired by the open source YOURLS and the mainstream URL shorteners available blingaer is designed to utilize the power of GAE to provide a unique scalable URL shortener service to the masses. With an integrated API external applications and/or users can easily communicate with blingaer to bring a simple and professional user experience.

Although development on this project has really only just begun a lot has been completed already and new ideas are still emerging. Eventually, the aim is to create a more generic Java engine that is not dependent on the GAE environment but for now concentration is definately on blingaer.
