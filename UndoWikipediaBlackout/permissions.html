The Google Chrome permissions mentioned on this page are required by the Undo Wikipedia Blackout extension to function.

Some of these permissions can be quite scary so I have attempted to explain why each one is required. If you doubt these explanations or are simply curious please feel free to take a look at the source code yourself. It is open source after all.

Visible

These permissions are displayed on the extension's page and on the warning message displayed to the user;

  • "Your data on neocotic.com" - This is required to for the inline installations process to update the Install buttons on the homepage once Undo Wikipedia Blackout has been successfully installed.
  • "Your data on wikipedia.org" - This is caused by the need to inject the script that prevents the blackout from being displayed.
  • "Your tabs and browsing activity" - This is caused by the need to inject the blackout prevention script in all previously opened pages on Wikipedia. This functionality is required when installing/updating the extension.
