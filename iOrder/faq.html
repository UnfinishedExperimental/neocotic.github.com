Why are the changes to my orders not being saved?

On the options page be sure that you always press "Save & Close" button whenever you want to save your changes. This applies even when you are only adding or deleting orders.

Why is there an error icon in the popup?

If there was a problem parsing an order's page you may see this error icon as well as any of the responsible orders highlighted. If you hover over the highlighted order label will see a short explanation of what caused the error. In most cases this will be due to incorrect order data being provided so check these on the options page (click the help buttons next to each field for more information).

However, due to the method in which iOrder retrieves order status updates it may be that the HTML structure of the order pages have changed and preventing the necessary values from being extracted. If you believe that this is the case please take the steps to raise this as an issue so that I can issue an update for this extension. This is not something I will easily notice as I'm not always buying iDevices from the Apple store.

Why do you need this permission...?

I've seen people commenting a lot on extensions in the Chrome Web Store about the permissions required by that extension. Most of these comments are negative and usually posted by people who do not fully understand the creation of Chrome extensions. For this reason I have created a breakdown of the permissions used by iOrder. However, if you have trust issues and don't trust that summary why not look at the code and see for yourself.

How can I get iOrder in a different language?

Currently, this extension is only provided in the English but I am hoping to support more languages in the future. I am currently working on a method of allowing others to easily submit translations and I will update this space when it is available.
