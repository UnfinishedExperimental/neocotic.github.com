The Google Chrome permissions mentioned on this page are required by the iOrder extension to function.

Some of these permissions can be quite scary so I have attempted to explain why each one is required. If you doubt these explanations or are simply curious please feel free to take a look at the source code yourself. It is open source after all.

Visible

These permissions are displayed on the extension's page and on the warning message displayed to the user;

  • "Your data on neocotic.com" - This is required to for the inline installations process to update the Install buttons on the homepage once iOrder has been successfully installed.
  • "Your data on store.apple.com" - This is required as iOrder requests your order's page from this website and parses the response to check for any status updates and to extract the tracking link.
  • "Your tabs and browsing activity" - This is required to prevent the Options link in the popup from opening a new tab each time it's clicked.

Further information on these permissions can be found here and here.

Hidden

Users are not informed about these permissions but are required by iOrder;

  • "notifications" - This is required to display notifications to the user (if the associated option is enabled).

Further information on these permissions can be found here.
