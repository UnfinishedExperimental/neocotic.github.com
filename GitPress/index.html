GitPress is a WordPress plugin that allows users to integrate information from their open source projects hosted on GitHub in to their blogs. The plugin provides widgets and shortcodes which you can customize to suit your own needs.

This project is still in the planning stage and development has not yet started but I'm hoping to very soon.
